# DS18B20

## Para que sirve

Sensor digital de temperatura con bus OneWire.

## Ventajas

- buena precision
- cableado simple
- se pueden poner varios en el mismo bus

## Consideraciones

- usar resistencia pull-up tipica de 4.7k en la linea de datos
- conversion lenta comparada con sensores analogicos

## Librerias comunes

- OneWire
- DallasTemperature

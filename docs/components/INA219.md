# INA219

## Para que sirve

Medicion de voltaje, corriente y potencia por I2C.

## Casos comunes

- monitoreo de bateria
- energia de motores pequenos
- consumo de cargas DC

## Consideraciones

- revisar rango de shunt y corriente maxima del modulo
- cuidar tierras compartidas
- calibrar si necesitas precision mejorada

## Libreria comun

- Adafruit INA219

## Salidas utiles

- bus voltage
- shunt voltage
- current
- power

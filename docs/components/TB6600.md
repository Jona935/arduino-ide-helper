# TB6600

## Para que sirve

Driver externo para motores paso a paso mas robusto que drivers pequenos tipo A4988.

## Cuando elegirlo

- motores con mas corriente
- aplicaciones industriales ligeras
- mesas, actuadores, CNC, automatizacion

## Consideraciones

- revisar microstepping y corriente con DIP switches
- requiere fuente externa correcta
- usar tiempos STEP/DIR compatibles con el driver

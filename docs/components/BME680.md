# BME680

## Para que sirve

Sensor ambiental con temperatura, humedad, presion y gas.

## Ideal para

- calidad de aire basica
- estaciones ambientales
- nodos IoT

## Consideraciones

- normalmente trabaja a 3.3V
- la medicion de gas requiere calentamiento interno
- conviene filtrar y muestrear a cadencia estable

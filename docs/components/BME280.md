# BME280

## Para que sirve

Sensor ambiental para temperatura, humedad y presion.

## Interfaces

- I2C
- SPI

## Recomendacion rapida

- perfecto para estaciones ambientales, dataloggers y monitoreo IoT
- usar I2C cuando quieras wiring simple

## Consideraciones

- muchos modulos son 3.3V
- verificar si la breakout ya trae regulador y level shifting
- humedad y temperatura requieren tiempo de estabilizacion

## Librerias comunes

- Adafruit BME280 Library
- Adafruit Unified Sensor

## Salida esperada

- temperatura en C
- humedad relativa en %
- presion en hPa o Pa
- altitud estimada opcional

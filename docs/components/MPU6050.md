# MPU6050

## Para que sirve

IMU de 6 ejes: acelerometro + giroscopio.

## Interfaces

- I2C

## Consideraciones

- ruido y drift requieren filtrado
- recomendable calibrar offsets al inicio
- para angulos estables conviene filtro complementario o Kalman simple

## Casos comunes

- estabilizacion
- inclinacion
- deteccion de movimiento
- robotica
